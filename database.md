# Database

- `show databases;`
- `create database fedProject;`
- `use fedProject;`
- ``` SQL
    create table articles (
        id integer primary key auto_increment,
        subject varchar(255),
        title varchar(255),
        description varchar(255),
        date timestamp default current_timestamp,
        text text(65535) ); 
    ```
